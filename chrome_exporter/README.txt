Cet petite extension chrome a pour but de pouvoir exporter en un click Cookies, historique et favoris.
Elle les exporte sous forme de JSON.
Amélioration à venir:
-Pouvoir importer les JSON afin de pouvoir retrouver ses favoris histo et cookies!
-Exporter un fichier supplementraire contenant uniquement le nom des cookies, les url de lhistorique et des favoris sous forme plus digeste que ce bon vieux JSON.


Pour l'installer sur votre chrome, 2 solutions:
1)Téléchargez le dossier de l'extension, allez sur chrome://extensions/ 
activez le mode développeur de votre chrome choisissez "chargez l'extension non empaquetée" et sélectionnez le dossier que vous venez de télécharger.

2)
Téléchargez le fichier crx
allez sur chrome://extensions/
glissez le fichier crx

Utilisation:
vous cliquez sur l'icone de l'app qui est dans le navigateur en haut à droite et vous pouvez choisir 
d'exporter votre historique, cookies et favoris sous forme de json!
